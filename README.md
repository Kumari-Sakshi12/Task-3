🌐 NexHub Workspace

A modern multi-page web application built using HTML, CSS, and JavaScript. This project combines three interactive features into one responsive website:

- 🌦 Live Weather App
- 📝 Programming Quiz
- 😂 Random Joke Generator

---

📌 Features

🏠 Home Page

- Attractive landing page
- Responsive banner image
- Feature cards
- Easy navigation

🌦 Weather App

- Search weather by city name
- Displays:
  - Temperature
  - Weather condition
  - Description
  - Feels like
  - Humidity
  - Pressure
  - Wind speed
- Uses OpenWeather API

📝 Programming Quiz

- Multiple-choice programming questions
- Instant answer validation
- Highlights correct and incorrect answers
- Displays final score after completion

😂 Random Joke Generator

- Fetches random jokes using an API
- Changes image every time a new joke is generated

🚀 How to Run

1. Download or clone the project.
2. Open the project folder.
3. Open index.html in your browser.
4. Enter a city name in the Weather section.
5. Play the Programming Quiz.
6. Generate unlimited random jokes.
